## Install frontend with yarn

Go to the frontend directory

```bash
  cd frontend/
```

Install dependencies
```bash
  yarn
```

Start Project [ Local ]
```bash
  yarn start
```

Start Project [ Dev ]
```bash
  yarn start_stg
```

Start Project [ Production ]
```bash
  yarn start_prod
```

## Build frontend with yarn
Go to the frontend directory

```bash
  cd frontend/
```

Build Project Dev
```bash
  yarn build_stg
```

Build Project Production
```bash
  yarn build_prod
```
